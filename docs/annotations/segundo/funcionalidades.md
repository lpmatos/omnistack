# Semana Omnistack - Segundo dia

## Pensando nas entidades e funcionalidades

Se o layout da aplicação já está disponível, geralmente utilizar isso para pensar no nosso backend. Agora se você precisar realmente pensar na aplicação, isso já é um pouco diferente.

### Entidades

As entidades é tudo aquilo que representa algo que será salvo no Banco de Dados.

* ONG.
* Incident (caso).

Uma ONG pode ter vários casos.

### Funcionalidades

* Login de ONG.
* Logout de ONG.
* Cadastro de ONG.
* Cadastro de novos indicentes (casos).
* Deletar incidentes (casos).
* Listar casos específicos de uma ONG.
* Listar todos os casos.
* Entrar em contato com a ONG (wpp e email).
